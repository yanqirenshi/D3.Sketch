# D3.Sketch

# Usage

`./src/sketch.js` を変更しながら、D3.js の描画とそのデータを作り込んでいきます。


# Setup

`./src/index.html` をブラウザで開くだけで動きます。

# Dependency

- [D3.js](https://d3js.org/)
- [Bulma](https://bulma.io/)
